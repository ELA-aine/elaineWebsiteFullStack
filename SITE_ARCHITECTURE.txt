DATA FLOW:

State Management: React State (useState hooks)
├── Game scores kept in component state
├── Form data stored in About.js state
├── Game configurations in each game component
└── Navigation handled by React Router

No External Database - Everything works offline!
Can add localStorage/backend later if needed.

STATE MANAGEMENT SUMMARY:

Home.js         - No complex state
Projects.js     - Static project data
Photography.js  - [selectedImage] for lightbox
Music.js        - No complex state
About.js        - [formData] for contact form, [submitted] for feedback
Interactive.js  - [activeGame] to switch between games
DrawingCanvas   - [color], [pixelSize], canvas ref
MemoryGame      - [cards], [flipped], [matched], [moves], [gameWon]
PixelPiano      - [activeKey] for visual feedback
QuoteGenerator  - [currentQuote], [displayedText], [isTyping]
ClickCounter    - [clicks], [level], [clickValue], [autoClicker], etc.
MazeGame        - [playerPos], [goalPos], [maze], [won], [moves]

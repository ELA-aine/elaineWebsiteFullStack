🎨 CHANGE COLORS (Optional):

Want different colors?

1. Open: frontend/src/styles.css
2. Find: Lines 3-9 (CSS Variables)
3. Change color codes:

   --primary-color: #ff006e;      ← Change this
   --secondary-color: #8338ec;    ← Or this
   --accent-color: #ffbe0b;       ← Or this

Example color palettes:

NEON: #ff006e (pink), #8338ec (purple), #ffbe0b (yellow)
RETRO: #ff0000 (red), #00ff00 (green), #0000ff (blue)
CYBERPUNK: #00ff88 (lime), #ff00ff (magenta), #00ffff (cyan)

4. Save and see instant changes!

  --dark-bg: #0a0e27;           ← Background (currently dark blue)
  --light-text: #e0e0e0;        ← Text (currently light gray)

🎮 GAME FEATURES:

PIXEL CANVAS
- Draw on grid with pixel brush
- Customize pixel size (4px, 8px, 12px, 16px)
- Color picker
- Download artwork as PNG
- Clear canvas button

MEMORY GAME
- 16 cards with 8 emoji pairs
- Track moves
- Win when all matched
- New Game button

PIXEL PIANO
- 7 notes (C-B)
- Click buttons or press keyboard
- 8-bit synth sound
- Visual feedback

QUOTE GENERATOR
- 15 inspirational quotes
- Typewriter animation
- Author attribution
- Get next quote

CLICK COUNTER
- Click character to earn points
- 3 upgrades:
  - Click Power ($50) - More points per click
  - Auto-Clicker ($1,000) - Passive income
  - Multiplier ($5,000) - Multiply all earnings
- Level system
- Reset game

MAZE GAME
- 10x10 randomly generated
- Arrow keys to navigate
- Reach yellow square to win
- Move counter
- New Maze button
